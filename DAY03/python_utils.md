List methods

nums = [1,2,3]

append() => nums.append(4) => Add item to list
extend() => nums.extend([5,6]) => Add multiple items
insert() => nums.insert(1, 10) => insert at index
remove() => nums.remove(2) => remove first occurence
pop() => nums.pop() => removes and returns last item
pop(i) => nums.pop(1) => removes item at index
index() => nums.index(3) => find position of item
